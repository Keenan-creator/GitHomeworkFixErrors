## 2. What is JavaScript?

JavaScript is a dynamic programming language, primarily used to add interactivity and dynamic content to web pages.

## 3. What three problems does Git & GitHub solve?

Enabling collaboration, version control, and streamlined workflows in Software development.

## 4. What happens when you `fork` a repository?

Forking lets you make changes to a project without affecting the original repository by copying a version under your repository name.

## 5. What happens when you clone a repository?

This allows you to make changes on the version of the software on your local machine.

## 6. What is a Pull Request?

A pull request is a proposal to merge, where collaborators can review and discuss the proposed set of changes before they integrate the changes into the main codebase.
